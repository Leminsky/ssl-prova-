O OpenSSL é um conjunto de ferramentas (linhas de comando) e bibliotecas usadas para implementar e utilizar criptografia e comunicação segura na internet.
Ele é amplamente usado para trabalhar com TLS/SSL (o “cadeado” do HTTPS), além de permitir tarefas como geração e validação de certificados digitais, criação de chaves criptográficas, cálculo de hashes
e verificação de assinaturas digitais.
Neste exercício, o OpenSSL foi instalado via apt e a instalação foi validada com openssl version, que confirma que o pacote está disponível no sistema e informa a versão instalada. 
Em ambientes reais, o OpenSSL é essencial para proteger conexões e dados, garantindo principalmente:
Confidencialidade: os dados trafegam criptografados.
Integridade: os dados não são alterados sem detecção.
Autenticidade: certificados ajudam a confirmar a identidade do servidor/serviço.
Em resumo, o OpenSSL é uma ferramenta fundamental para administrar e diagnosticar recursos de segurança baseados em TLS/SSL e criptografia em sistemas Linux
