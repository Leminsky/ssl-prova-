O OpenSSL é um conjunto de ferramentas (linhas de comando) e bibliotecas usadas para implementar e utilizar criptografia e comunicação segura na internet.<br>
Ele é amplamente usado para trabalhar com TLS/SSL (o “cadeado” do HTTPS), além de permitir tarefas como geração e validação de certificados digitais, criação de chaves<br> criptográficas, cálculo de hashes<br>
e verificação de assinaturas digitais.<br>
Neste exercício, o OpenSSL foi instalado via apt e a instalação foi validada com openssl version, que confirma que o pacote está disponível no sistema e informa a versão<br> instalada. <br>
Em ambientes reais, o OpenSSL é essencial para proteger conexões e dados, garantindo principalmente:<br>
Confidencialidade: os dados trafegam criptografados.<br>
Integridade: os dados não são alterados sem detecção.<br>
Autenticidade: certificados ajudam a confirmar a identidade do servidor/serviço.<br>
Em resumo, o OpenSSL é uma ferramenta fundamental para administrar e diagnosticar recursos de segurança baseados em TLS/SSL e criptografia em sistemas Linux<br>
